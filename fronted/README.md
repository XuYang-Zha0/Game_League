# 电子竞技赛事数据融聚平台（初始版本）

本项目是毕业设计的前端初始系统，目标是将 **CS2 / 无畏契约 / 英雄联盟** 三个项目的赛事、战队、选手等数据整合在同一个网站中，并支持统一检索、筛选、展示与导出。

## 已实现内容

- 三项目切换：`CS2`、`VALORANT`、`LOL`
- 异构数据适配：不同字段结构统一映射为前端通用模型
- 核心业务模块：排行榜、赛事、比赛、战队、选手
- 智能分析模块：比赛摘要、关键转折点、战队/选手分析、结构化输出
- 基础支撑模块：关键词检索、赛区筛选、赛事级别筛选、JSON 导出
- 架构展示：数据来源层 → 数据处理层 → 数据存储层 → 应用分析层

## 目录说明

```text
src/
  data/
    rawSources.js      # 各游戏原始 mock 数据（字段不统一）
    adapters.js        # 字段标准化与统一模型映射
    platformService.js # 数据读取、分层信息、导出数据生成
  App.vue              # 主界面（模块组合与交互逻辑）
  style.css            # 主题样式与响应式布局
```

## 运行方式

```bash
npm install
npm run dev
```

构建生产包：

```bash
npm run build
```
